## 🌟 一、为什么要先了解钱包与测试币？

在区块链世界里，**钱包就是你的“身份证 + 银行账户”**。  
它保存你的地址（0x 开头）、签名交易、访问去中心化应用（DApp）。

但刚开始我们不想用真钱测试，于是就用“测试网络（Testnet）”和“测试币（Fake ETH）”。

- **主网（Mainnet）**：真币、真交易（有风险）
    
- **测试网（Testnet）**：假币、假交易（练习用）
    

---

## 🦊 二、安装钱包（MetaMask）

### 📱 步骤 1：下载并安装

1. 前往官网 👉 [https://metamask.io](https://metamask.io/)
    
2. 选择安装 Chrome 插件或手机 App（App Store / Google Play）
    
3. 安装后点击小狐狸图标 🦊
    

### 🪄 步骤 2：创建钱包

1. 点击「Create a Wallet」
    
2. 设置密码（建议 ≥8 位）
    
3. **记录好助记词（Seed Phrase）**
    
    - 助记词是你的唯一钥匙 🔑
        
    - 别截图！别保存到云端！
        
    - 抄写到纸上，离线保存。
        

> ⚠️ 小贴士：任何要求你“输入助记词”的网站，**100%是骗子！**

---

## 🔄 三、切换到测试网络

1. 打开 MetaMask，点击顶部网络下拉框（默认显示 _Ethereum Mainnet_）
    
2. 点击「Show Test Networks」
    
3. 选择 **Sepolia Test Network**
    

💡 “Sepolia” 是以太坊的主力测试网，几乎完全模拟主网环境。

---

## 💧 四、获取测试币（Sepolia ETH）

测试币 = 免费的“假以太坊”，让你在测试网上练习交易、部署合约。

---

### ✅ 推荐方法一：Google 官方 Faucet（最稳定）

1. 打开网站 👉 [https://cloud.google.com/application/web3/faucet/ethereum/sepolia](https://cloud.google.com/application/web3/faucet/ethereum/sepolia)
    
2. 粘贴你的钱包地址（以 0x 开头）
    
3. 点击「Request funds」
    
4. 稍等几秒，你就能看到提示：
    
    > ✅ Drip complete! Testnet tokens sent!
    
5. 打开钱包刷新，就能看到约 0.05 SepoliaETH。
    

---

### ✅ 推荐方法二：ChainDrop Faucet（备用）

1. 访问 👉 [https://chaindrop.org/?chainid=11155111](https://chaindrop.org/?chainid=11155111)
    
2. 点击「Connect Wallet」
    
3. 确认钱包连接到 **Sepolia**
    
4. 点击「Request from Faucet」
    
5. 稍等几分钟，钱包余额 +0.05 SepoliaETH 💧
    

---

## 🔍 五、查看你的链上记录

打开区块浏览器：[https://sepolia.etherscan.io](https://sepolia.etherscan.io/)  
粘贴你的钱包地址（0x 开头那串），  
就能看到刚才那笔交易记录 ✨

![](https://etherscan.io/images/brandassets/etherscan-logo-circle.svg)

> 💬 这代表你已经在区块链上留下了**第一笔可追溯记录**！

## ⚙️ 六、下一步：尝试部署你的第一个智能合约！

现在你已经拥有钱包 + 测试币，可以尝试在 [https://remix.ethereum.org](https://remix.ethereum.org/)  
部署一个「Hello Web3!」合约：

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract HelloWorld {
    string public message = "Hello Jo 💫";

    function setMessage(string memory newMessage) public {
        message = newMessage;
    }
}
```

点击 **Compile → Deploy → Confirm**，  
几秒后你会在 Etherscan 上看到属于你自己的合约地址 💎
